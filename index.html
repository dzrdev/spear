📋 Open Graph 설정

이 페이지는 다음 메타 태그를 포함합니다:

<meta property="og:title" content="Spear - Image Upload Test"> <meta property="og:description" content="..."> <meta property="og:image" content="..."> <meta property="og:url" content="...">

디스코드에서 이 링크를 공유하면 위의 정보가 임베드로 표시됩니다.

🎨 이미지 변경 방법

<meta property="og:image"> 태그의 content 값을 변경하세요:

<meta property="og:image" content="YOUR_IMAGE_URL">

예시:

  • 절대 URL: https://example.com/image.jpg
  • 리포지토리 이미지: https://raw.githubusercontent.com/dzrdev/spear/main/images/preview.jpg
  • GitHub Pages 이미지: https://dzrdev.github.io/spear/images/preview.jpg

최소: 200 x 200px
